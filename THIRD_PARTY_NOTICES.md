# Third-Party Notices

This repository includes project components that were developed with reference to third-party open-source work.

## GluPred

The glucose prediction part of this project was developed with reference to:

- Project: `r-cui/GluPred`
- Repository: https://github.com/r-cui/GluPred
- Paper: "Personalised Short-Term Glucose Prediction via Recurrent Self-Attention Network"
- Original license: MIT

This repository adapts ideas and parts of the prediction-related implementation and workflow for integration into the GlucoPI graduation project.

## Dataset

The glucose prediction workflow uses the OhioT1DM dataset.

- Dataset name: OhioT1DM Dataset
- Source referenced by GluPred: http://smarthealth.cs.ohio.edu/OhioT1DM-dataset.html
