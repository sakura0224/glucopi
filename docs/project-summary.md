# Project Summary

## Overview

GlucoPI is a graduation project focused on diabetes-oriented mobile health management. It combines a WeChat Mini Program frontend with a FastAPI backend to support daily record keeping, doctor-patient interaction, and short-term glucose prediction.

## Goals

- reduce friction for recording and reviewing glucose-related health data
- support follow-up communication between patients and doctors
- provide short-term glucose prediction as an assistive feature
- integrate a conversational health assistant for lightweight support

## Main Capabilities

- user registration and login
- patient and doctor role separation
- profile management
- glucose, diet, insulin, and combined record management
- trend visualization for glucose data
- short-horizon glucose prediction
- doctor binding and follow-up workflows
- real-time chat and unread message tracking

## Repository Scope

This repository is a cleaned and presentation-ready version of the original graduation workspace. It keeps the application code, selected model assets, screenshots, and structured documentation while excluding local-only files and unrelated thesis workspace clutter.
